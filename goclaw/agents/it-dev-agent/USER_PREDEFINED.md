# USER_PREDEFINED — IT/Dev Agent

## Người dùng agent này

Toàn bộ nhân viên (báo lỗi IT) + dev team (technical tasks) + IT admin (infra/access).

## Profile chung

- **Timezone:** Asia/Ho_Chi_Minh (UTC+7)
- **Ngôn ngữ:** Tiếng Việt (report lỗi), English (technical terms, logs, code)
- **Giờ support:** 8:00–18:00 Thứ Hai–Thứ Sáu
- **Kênh:** Telegram DM (ưu tiên), ticket system

## Nhóm người dùng

| Nhóm | Quyền | Use cases |
|---|---|---|
| Nhân viên | Báo lỗi | Máy tính lỗi, phần mềm, mạng, cấp quyền |
| Dev | Technical requests | Bug fix, code review, deploy, infra |
| IT Admin | Full access | Server, access control, security |
| Sếp | Overview | System health, incident reports |

## Severity levels

- **P0 — Critical:** Hệ thống sập, toàn bộ team không làm được. Response < 15 phút.
- **P1 — High:** Nhiều người bị ảnh hưởng, block công việc chính. Response < 1 giờ.
- **P2 — Medium:** Một người bị ảnh hưởng, có workaround. Response < 4 giờ.
- **P3 — Low:** Yêu cầu nhỏ, không block. Response < 1 ngày làm việc.
