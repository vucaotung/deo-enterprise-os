# USER_PREDEFINED — Office Admin Agent

## Người dùng agent này

Toàn bộ nhân viên (yêu cầu hành chính) + admin team (xử lý yêu cầu) + sếp (phê duyệt).

## Profile chung

- **Timezone:** Asia/Ho_Chi_Minh (UTC+7)
- **Ngôn ngữ:** Tiếng Việt
- **Giờ làm việc:** 8:00–18:00 Thứ Hai–Thứ Sáu
- **Kênh:** Telegram DM hoặc group admin

## Nhóm người dùng

| Nhóm | Quyền | Use cases |
|---|---|---|
| Nhân viên | Tạo yêu cầu | Đặt phòng, mua sắm, văn phòng phẩm, sửa chữa |
| Admin | Xử lý yêu cầu | Duyệt, mua, cấp phát, theo dõi |
| Sếp | Phê duyệt lớn | Mua sắm > ngưỡng, hợp đồng cung cấp |

## SLA xử lý yêu cầu

| Loại yêu cầu | SLA |
|---|---|
| Văn phòng phẩm cơ bản | 1 ngày làm việc |
| Đặt phòng họp | Ngay lập tức (nếu còn trống) |
| Mua sắm thiết bị < 5 triệu | 3 ngày làm việc |
| Mua sắm thiết bị > 5 triệu | 5 ngày làm việc (cần phê duyệt) |
| Sửa chữa cơ sở vật chất | 2 ngày làm việc |
