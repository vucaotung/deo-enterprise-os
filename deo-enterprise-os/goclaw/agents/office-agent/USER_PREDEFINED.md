# USER_PREDEFINED — Office Agent

## Người dùng agent này

Staff tạo tài liệu, báo cáo, và file văn phòng. Bao gồm management, admin, và bất kỳ ai cần soạn văn bản chuyên nghiệp.

## Profile chung

- **Timezone:** Asia/Ho_Chi_Minh (UTC+7)
- **Ngôn ngữ:** Tiếng Việt (văn bản), English (tên file, thuật ngữ kỹ thuật)
- **Output:** File lưu local hoặc Google Drive

## Defaults khi không có chỉ định

- **Khổ giấy:** A4
- **Font mặc định:** Times New Roman (văn bản hành chính), Calibri (báo cáo doanh nghiệp)
- **Ngôn ngữ văn bản:** Tiếng Việt
- **Format số:** VND, dấu chấm ngăn nghìn (1.000.000)
- **Format ngày:** DD/MM/YYYY
- **Palette mặc định:** Đen trắng (hỏi khi cần màu)

## Văn bản thường dùng

- Công văn, tờ trình, biên bản (NĐ 30)
- Đề xuất, báo cáo nội bộ
- Bảng theo dõi tiến độ (Excel)
- Báo cáo trình bày (PowerPoint)
